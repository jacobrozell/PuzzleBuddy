Accessibility

Puzzle Buddy for iOS

Puzzle Buddy is committed to making the app usable for everyone. We follow WCAG 2.1 Level AA as a target for the iOS app and work toward full conformance over time.

Conformance status

Partial conformance with WCAG 2.1 Level AA. Core flows (puzzle list, add puzzle, settings) include VoiceOver support and respect system accessibility settings. Account sign-in is planned for a future release; login screens remain tested behind a feature flag. Some screens — including custom rating controls — are still being improved.

Supported features

  • VoiceOver — Labels on puzzle list, add puzzle, settings, and tab navigation; login fields when sign-in is enabled
  • Dynamic Type — System text sizes via standard iOS fonts
  • Dark Mode — Adaptive colors for light and dark appearance
  • Reduce Motion — Static background instead of decorative gradient when Reduce Motion is enabled
  • Sign in with Apple — Available when account sign-in ships; uses system accessibility affordances

Known limitations

  • Layout at the largest accessibility text sizes is not yet fully verified on all screens
  • The app is currently available in English only

WCAG 2.1 principles we apply

Principle What it means in Puzzle Buddy
Perceivable Text alternatives and labels for controls; sufficient color contrast on primary actions; support for larger text
Operable Touch and VoiceOver navigation; no time limits; reduced motion option for decorative animation
Understandable Clear field labels, consistent tab navigation, error messages in plain language
Robust Standard iOS controls and accessibility APIs so assistive technology can interpret the interface

Testing with assistive technology

On your iPhone or iPad, open Settings → Accessibility to enable:

  • VoiceOver — Spoken descriptions of on-screen elements
  • Larger Text — Increase text size under Display & Text Size
  • Reduce Motion — Limits non-essential animation
  • Voice Control — Navigate and activate controls by voice

Feedback

If you encounter a barrier when using Puzzle Buddy, please contact us via Support. Helpful details include:

  • Your iOS version
  • Which assistive feature you use (e.g. VoiceOver, Larger Text)
  • What you were trying to do and what happened instead

We review accessibility feedback regularly and prioritize fixes that block core tasks (adding puzzles, viewing your collection, settings).
